# Java Projects

## DE: Projektueberblick
Sammlung von Java-Lernprojekten vom Einstieg bis zu kleinen Anwendungsprojekten.
Schwerpunkt: OOP, Fehlerbehandlung, Datenstrukturen und praktische CLI-Programme.

## DE: Struktur
- `01_HelloWorldPlus` bis `10_ExceptionHandlingDemo`: Grundlagen bis Mittelstufe
- `11_LingoShield`: Terminologie- und Textpruefung
- `14_governance-analyzer`: groesseres Analyseprojekt mit Maven-Struktur
- `15_LagerApp`: Lagerverwaltung (in Arbeit)
- `12_Arduino_PullUp_PullDown`, `13_Arduino_Prototypes`: hardware-nahe Uebungen im Java-Lernumfeld

## DE: Lokal ausfuehren
```bash
# Beispiel fuer ein einzelnes Projekt
javac 02_SimpleCalculator/SimpleCalculator.java
java -cp 02_SimpleCalculator SimpleCalculator
```

## DE: Lernziele
- Saubere Java-Grundlagen (Klassen, Methoden, Kontrollfluss)
- Objektorientiertes Design mit wartbarer Struktur
- Schrittweise Richtung testbare, produktionsnahe Projekte

## EN: Overview
